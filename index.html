Surgimento da Internet

(Voltar)

A internet teve início com a chamada ARPANet, uma intranet da DARPA (Defense Advanced Research Projects Agency) no final da década de 50.

A ARPANet era um sistema de comunicação em pacotes enviados 1 a 1 e com múltiplos destinos, a rede servia para fins de pesquisas e a primeira conexão foi em 29 de Outubro de 1969 entre a UCLA e a SRI.

Como teste, foi enviada a palavra "login", mas o destinatário recebeu apenas "lo", o que pode ser considerado também a primeira queda de conexão da história.

A ARPANet passou a ser utilizada tanto por pesquisadores como militares e assim também surge o correio eletrônico (e-mail), permitindo a troca de mensagens entre usuários.

O uso da rede se tornou tão intenso que a intranet teve de ser desvinculada e em 1983 o acesso dos militares passa a ser pela Milnet, deixando a ARPANet apenas para fins de pesquisas científicas.

Em 1989, Tim Berners-Lee viu a oportunidade de unir hipertexto com o protocolo TCP/IP e assim surge a www (World Wide Web).

Como funciona o HTML

(Voltar)

Primeiramente, o HTML não é uma linguagem de programação, pois supõe-se que uma linguagem de programação seja capaz de resolver problemas, cálculos, criar aplicações entre outros e o HTML funciona apenas como ferramenta de marcação.

HTML, ou Hyper Text Markup Language é uma linguagem de marcação, onde textos são delimitados por nomes (ou tags) em que o navegador é capaz de interpretar.

Cada um destes delimitadores/tags possuem uma função e comportamento específicos.

O que são tags HTML e como funciona uma página de internet

(Voltar)

Em HTML, existem as chamadas tags, que realizam uma função específica dentro do documento. As tags funcionam utilizando os sinais matemáticos de menor (<) e maior (>) ao abri-las e ao fechá-las, porém para fechar é necessário inserir uma barra logo após o sinal de menor.

Por exemplo, as chamadas tags "h", que vão de h1 à h6, servem para delimitar o título, a tag h1 cria um título com letras grandes, a tag h2 com letras menores e assim por diante até a h6, criamos esta tag da seguinte maneira: < h1 > (texto) < /h1 > (sem espaços).

Uma página de internet é composta por diversas tags HTML, são elas que irão compor o "esqueleto" do site, enquanto uma outra linguagem chamada de CSS (Cascade Styling Sheets) será responsável pelas cores, diagramações, animações entre outros.

E além do HTML e CSS há o JavaScript, essa sim é uma linguagem de programação e será responsável pelo comportamento e funcionalidade do site, sendo responsável por enviar dados de formulários, voltar e avançar páginas, coletar dados de uma API, mostrá-los na tela entre outros.
